=== Datenbanksystem

=== Zahlungsportale  
* System benachrichtigen, dass Zahlung eingegangen ist (1) 

=== Emailportal 
* Bestellbestätigung erhalten und an den Kunden senden (1)
* über Werbeaktionen informieren (3)
